AirWatch Weather App

Project information

  • Category: Front-end Web App
  • Project timeline: Sep 2023 - Dec 2023
  • Project URL: Click here to view project on Github

  • React-based web app that utilizes the Weatherbit API to retrieve air quality details from over 50,000 live weather stations and displays them on a dashboard
  • Utilizes the Recharts data visualization library to showcase weather information using graphs and charts
