Online Room Reservation System

Project information

  • Category: Full-stack Web App
  • Project timeline: 2023 - 2024
  • Project URL: Click to view project on GitFront

  • React and Django-based online room reservation system with an efficient scheduling algorithm, user-friendly UI, and an integrated catering section for food orders
  • Collaborated with a cross-functional team of 8 and acted as the project manager, reducing the amount of back-and-forth calls and visits from potentially 100+ customers
  • Other key functionalities include automated booking confirmation emails, seamless Google Calendar integration for confirmed reservations, and generation of booking details forms for easy printing
