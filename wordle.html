Wordle iOS App

Project information

  • Category: iOS Application
  • Project timeline: March 2024
  • Project URL: Click here for permission to view on Github (Include your name, role and Github username)

  • iOS app that allows the user to guess a random, variable-length word in a fixed number of chances, providing hints based on the accuracy of letter placement
  • Features:
  • • User can change the number of guesses they can make, select a category, change the length of the word, reset the board without changing the target word, or enter alien mode which changes the target word after each guess!
  • • Different categories for questions
  • • User can change the length of the target word
  • • User can reset the board without changing the target word
  • • Alien mode which changes the target word after each guess!
