EduAccess Mobile App

Project information

  • Category: Android Application
  • Project timeline: 2021 - Present (new update in progress)
  • Project URL: Click here to view project on GitHub

  • Android app created with Html, CSS, Javascript and the Cordova Ionic Platform for the purpose of allowing students to access learning resources such as pdfs and videos offline
  • App helped over 65 students since the pandemic and is still in use today
  • Integrated with the Cordova file storage and the PDF Viewer plugin, allowing offline access to over 1,500 notes and worksheets
  • Currently enhancing the app using Flutter for cross-platform capability, improved UI, and increased efficiency
